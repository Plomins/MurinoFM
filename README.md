# Murino FM — Spring Boot REST API

Учебный проект по разработке серверного приложения для управления музыкальной библиотекой. 
## Суть проекта
Проект как замена "spotify" без цензуры,без платной подписки.
C свободным заливом музыки без особо строгой модерации в нем будет реализовано система рейтингово подбора музыки, где у каждого трека есть свои определеный значение по жанру и т.д.

## Технологии
* **Java 25** (OpenJDK) — проект использует последние возможности языка.
* **Spring Boot 4.0.3** — база приложения.
* **Spring Data JPA & H2** — работа с данными в памяти.

## Архитектура
Приложение построено по слоям (Layered Architecture):
* **Controller**: Прием HTTP-запросов.
* **Service**: Обработка логики и Stream API.
* **Repository**: Доступ к БД через JPA.
* **DTO/Entity**: Модели данных и объекты передачи.
